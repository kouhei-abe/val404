Hac&nbsp;quattuor partes possunt id est historia, tropologiam allegoria, anagogen, in uno conuenire, ut est,&nbsp;Hierusalem&nbsp;&nbsp;secundum historiam ciuitas iudaeorum. Secundum&nbsp;allegoriam&nbsp;aecclesia Christi. Secundum anagogen ciuitas dei illa caelestis, quae est mater omnium nostrum. Secundum tropologiam uero anima hominis quae frequenter&nbsp;hoc inter praetatur aut laudatur a domino. De his ergo apostolus, si uenero inquit as uos linguis loquens, quid uobis prodero nisi uobis prodero aut in reuelatione aut in scientia aut in prophetia aut in doctrina.
Practicam rationem iuxta quod ualuimus disseruimus, nunc as rertiam partem id est loicam disserendam properemus.
Multum laboramus sed utinam laboris dructum tandem perceperimus.
Loica id est rationalis trifario modo subdiuiditur. In dialecticam id est dispu(?)tatoria. In epidictitam id est demonstratiuam. In sophisticam id est fraudu(?)lentam et fictam.
